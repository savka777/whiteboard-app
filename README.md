## Description
Use your mobile devices or tables to draw on computer screen in real time. 


## 1: Project setup

```bash
$ npm install
```

## 2: Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
## 3: Scan QR Code
connect any device by scanning the QR Code to be able to draw in real time on screen
